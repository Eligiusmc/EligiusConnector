# EligiusConnector

El plugin definitivo de puente Discord-Minecraft para Bukkit, Spigot, Paper, Purpur y Folia.

## Características

- 🔗 **Vinculación de Cuentas:** Vinculación segura por código entre cuentas de Discord y Minecraft.
- 💬 **Chat Compartido:** Chat en tiempo real entre Minecraft y Discord.
- 🖥️ **Consola Remota:** Ejecuta comandos de consola desde Discord.
- 🔄 **Sincronización de Roles:** Sincronización bidireccional con LuckPerms.
- 📊 **Consultas de Jugadores:** Ver perfiles, inventario, ubicación desde Discord.
- 🎉 **Eventos Personalizados:** Eventos configurables con triggers de timer y comando.
- 🛡️ **Filtros de Chat:** Sistema de punición escalada.
- 📝 **Registro de Auditoría:** Registro completo de todas las acciones.
- 🎯 **PlaceholderAPI:** Integración completa con placeholders.
- 🌐 **i18n:** Soporte multi-idioma (EN, ES).
- 📊 **bStats:** Métricas anónimas de uso.

## Inicio Rápido

1. Descarga el último `EligiusConnector-x.x.x.jar` desde la página de [Releases](https://github.com/Eligiusmc/EligiusConnector/releases).
2. Colócalo en tu directorio `plugins/`.
3. ¡Inicia tu servidor!
4. Visita la [Configuración](/es/config/) para aprender a configurar.

## Requisitos

- **Java 21 LTS** o superior
- **Minecraft 1.21 - 26.1.2+**
- **Paper, Spigot, Bukkit, Purpur o Folia**
- **Bot de Discord** (con permisos adecuados)
